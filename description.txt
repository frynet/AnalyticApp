Introduction
------------

The tablexplore package provides a desktop application for
plotting and tabular data analysis. It is implemented in PySide2 for Qt
with the pandas DataFrame class as the table model backend. You can re-use the
`DataFrameWidget` in your own PySide2 applications.

Installation
------------

::
  pip install -e git+https://github.com/dmnfarrell/tablexplore.git#egg=tablexplore
